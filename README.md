# 💀 HollowR00t's Machine Hacking Vault

Welcome to the source code of my Offensive Security Grimoire. 
This repository contains the Markdown files, assets, and automated pipelines used to generate my public portfolio of CTF writeups and vulnerability research.

## 🔗 Live Grimoire
The actual documentation is automatically compiled and hosted via GitHub Pages.
**Access the live vault here:** [https://hollowr00t.github.io/MachineHacking/](https://hollowr00t.github.io/MachineHacking/)

## ⚙️ Architecture (Docs-as-Code)
* **Framework:** MkDocs (Material Theme).
* **CI/CD:** GitHub Actions (Automated build and deployment to the `gh-pages` branch).
* **Format:** Markdown + Obsidian native callouts.
