  PORT     STATE SERVICE REASON  VERSION

Task 1 Which TCP port is open on the machine?
  6379 

Task 2 Which service is running on the port that is open on the machine?
  redis 

What is Redis?
Redis (REmote DIctionary Server) is an open-source advanced NoSQL key-value data store used as a
database, cache, and message broker.

We install redis-tools
  sudo apt install redis-tools
Then use redis-cli --help
We see -h <hostname> Server hostname (default: 127.0.0.1).

And use redis-cli -h 10.129.75.127
  redis-cli -h 10.129.75.127
  10.129.75.127:6379>

We use the command 'info' to see all that we need
